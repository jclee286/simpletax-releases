핵심 기능

거래 입력 + 자동 분개

수동 입력·엑셀 일괄 업로드. 적요로 학습된 규칙이 자동 분류 (4대보험·세금·이자수익 등 15개 기본 규칙 포함).

K-GAAP 표준 계정과목

한국 일반기업회계기준 표준 계정과목 72개 기본 시드. 손익계산서·재무상태표 자동 산출.

엑셀 저장 → 세무사 전달

월·분기·연 단위로 거래 내역을 엑셀 파일로 저장. 세무사가 분개·재무제표 작성에 그대로 활용.

100% 로컬 SQLite

회계 데이터는 사용자 PC 안에만 저장. 클라우드 업로드 없음. 1 PC = 1 회사 전제.

은행 엑셀 자동 변환

국민·우리은행 거래내역 .xls 자동 인식 + 표준 양식 변환. 다른 은행은 표준 양식 다운로드.

회원 가입 없음

설치 후 회사 정보 한 번 입력하면 바로 사용. 비밀번호·로그인 없음.

설치 방법

Windows

  1. simpletax-setup.exe 다운로드 후 더블클릭
  2. "Windows의 PC 보호" 경고 시 → "추가 정보" → "실행"
  3. 설치 마법사 진행 → 바탕화면 단축키로 실행
  4. 최초 실행 시 회사 정보 입력 → 바로 사용
SmartScreen 경고: 아직 EV 코드서명 인증서를 도입하지 않아 알 수 없는 게시자 경고가 뜰 수 있습니다. 베타 매출 발생 후 정식 코드서명을 도입할 예정입니다.

macOS (Apple Silicon)

  1. simpletax-mac.dmg 다운로드 후 더블클릭
  2. 열린 창에서 simpletax 아이콘 → Applications 폴더로 드래그
  3. 응용프로그램 폴더의 simpletax를 우클릭 → "열기" (첫 실행에만)
  4. "확인되지 않은 개발자" 경고에 "열기" 버튼 클릭
  5. 최초 실행 시 회사 정보 입력 → 바로 사용
Gatekeeper 우회: 아직 Apple Developer 코드서명을 도입하지 않아 처음 실행 시 우클릭→열기가 필요합니다. 한 번만 우회하면 그 후엔 정상 더블클릭으로 실행됩니다. Apple Silicon(M1~M4) 전용.
