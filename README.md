## 무엇인가요

- **1 PC = 1 회사** — 회계 데이터는 사용자 PC 안에만 (SQLite)
- **회원 가입 없음** — 설치 후 회사 정보만 입력하면 바로 사용
- **K-GAAP 표준** — 한국 일반기업회계기준 계정과목 72개 기본 시드
- **세무사 전달용** — 거래 내역을 엑셀로 저장 → 세무사가 분개·재무제표 작성에 활용
